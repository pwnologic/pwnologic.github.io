pecrack/pecan

Status: Work in Progress

Open-source IoT security scanner with a modern self-correcting approach.

pecrack is our specialized hardware platform that runs the pecan software suite to enable comprehensive IoT device assessment. While pecan provides the intelligence and exploitation capabilities, pecrack delivers the physical interfaces and hardware manipulation tools needed to directly interact with target devices. This hardware-software combination allows security researchers to perform complete security evaluations from both logical and physical perspectives.

pecrack can be used both invasively (direct connection to internal hardware) or via USB. Functions include:

USB Attacks

  • DMA exploitation to bypass OS protections
  • Controller firmware modification
  • Device spoofing (keyboard, storage, network)
  • Protocol fuzzing and exploitation

Firmware Operations

  • BIOS/UEFI extraction and modification
  • Secure boot bypass techniques
  • Flash memory reading/writing
  • Bootloader manipulation

Hardware Interfaces

  • JTAG/boundary scan access
  • UART/serial console interception
  • SPI/I2C bus monitoring
  • Debug port utilization

Side-Channel Methods

  • Power analysis during operations
  • Timing attack implementation
  • Voltage glitching for security bypass
  • Fault injection techniques

Security Assessment

  • Hardware configuration verification
  • Tamper protection evaluation
  • Firmware security analysis
  • Hardware vulnerability scanning
