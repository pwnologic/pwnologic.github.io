pecan

Status: Work in Progress

Open-source IoT security scanner with a modern self-correcting approach.

pecan is a lightweight command-line tool designed to identify and assess IoT devices on local networks. Unlike traditional security scanners, pecan employs adaptive scanning techniques that adjust based on device responses and network conditions.

Key Features:

  • Automated discovery of IoT devices across various protocols
  • Default credential checking against common IoT devices
  • Firmware version identification and vulnerability correlation
  • Encrypted communications analysis
  • Self-correcting scan methodologies that minimize false positives

Usage:

$ pecan scan --network 192.168.1.0/24
$ pecan device --ip 192.168.1.15 --deep
$ pecan report --format json --output results.json
