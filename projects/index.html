pecrack/pecan

Status: Work in Progress

Open-source IoT security scanner with a modern self-correcting approach.

pecrack is our specialized hardware platform that runs the pecan software suite to enable comprehensive IoT device assessment. While pecan provides the intelligence and exploitation capabilities, pecrack delivers the physical interfaces and hardware manipulation tools needed to directly interact with target devices. This hardware-software combination allows security researchers to perform complete security evaluations from both logical and physical perspectives.
